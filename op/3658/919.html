{ 14 - 15 } { وَلَوْ فَتَحْنَا عَلَيْهِمْ بَابًا مِنَ السَّمَاءِ فَظَلُّوا فِيهِ يَعْرُجُونَ * لَقَالُوا إِنَّمَا سُكِّرَتْ أَبْصَارُنَا بَلْ نَحْنُ قَوْمٌ مَسْحُورُونَ } .
أي: ولو جاءتهم كل آية عظيمة لم يؤمنوا وكابروا { وَلَوْ فَتَحْنَا عَلَيْهِمْ بَابًا مِنَ السَّمَاءِ } فصاروا يعرجون فيه، ويشاهدونه عيانا بأنفسهم لقالوا من ظلمهم وعنادهم منكرين لهذه الآية: { إِنَّمَا سُكِّرَتْ أَبْصَارُنَا } أي: أصابها سكر وغشاوة حتى رأينا ما لم نر، { بَلْ نَحْنُ قَوْمٌ مَسْحُورُونَ } أي: ليس هذا بحقيقة، بل هذا سحر، وقوم وصلت بهم الحال إلى هذا الإنكار، فإنهم لا مطمع فيهم ولا رجاء، ثم ذكر الآيات الدالات على ما جاءت به الرسل من الحق فقال:
