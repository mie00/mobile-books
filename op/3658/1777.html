{ 13-14 } { إِنَّ الَّذِينَ قَالُوا رَبُّنَا اللَّهُ ثُمَّ اسْتَقَامُوا فَلا خَوْفٌ عَلَيْهِمْ وَلا هُمْ يَحْزَنُونَ * أُولَئِكَ أَصْحَابُ الْجَنَّةِ خَالِدِينَ فِيهَا جَزَاءً بِمَا كَانُوا يَعْمَلُونَ }
أي: إن الذين أقروا بربهم وشهدوا له بالوحدانية والتزموا طاعته
[ ص 781 ]
وداموا على ذلك، و { اسْتَقَامُوا } مدة حياتهم { فَلا خَوْفٌ عَلَيْهِمْ } من كل شر أمامهم، { وَلا هُمْ يَحْزَنُونَ } على ما خلفوا وراءهم.
{ أُولَئِكَ أَصْحَابُ الْجَنَّةِ } أي: أهلها الملازمون لها الذين لا يبغون عنها حولا ولا يريدون بها بدلا { خَالِدِينَ فِيهَا جَزَاءً بِمَا كَانُوا يَعْمَلُونَ } من الإيمان بالله المقتضى للأعمال الصالحة التي استقاموا عليها.
