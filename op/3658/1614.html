{ 45 - 47 } { وَاذْكُرْ عِبَادَنَا إبْرَاهِيمَ وَإِسْحَاقَ وَيَعْقُوبَ أُولِي الأيْدِي وَالأبْصَارِ * إِنَّا أَخْلَصْنَاهُمْ بِخَالِصَةٍ ذِكْرَى الدَّارِ * وَإِنَّهُمْ عِنْدَنَا لَمِنَ الْمُصْطَفَيْنَ الأخْيَارِ } .
يقول تعالى: { وَاذْكُرْ عِبَادَنَا } الذين أخلصوا لنا العبادة ذكرا حسنا، { إِبْرَاهِيمَ } الخليل { و } ابنه { إِسْحَاقَ وَ } ابن ابنه { يَعْقُوبَ أُولِي الأيْدِي } أي: القوة على عبادة اللّه تعالى { وَالأبْصَار } أي: [ ص 715 ] البصيرة في دين اللّه. فوصفهم بالعلم النافع، والعمل الصالح الكثير.
{ إِنَّا أَخْلَصْنَاهُمْ بِخَالِصَةٍ } عظيمة، وخصيصة جسيمة، وهي: { ذِكْرَى الدَّارِ } جعلنا ذكرى الدار الآخرة في قلوبهم، والعمل لها صفوة وقتهم، والإخلاص والمراقبة للّه وصفهم الدائم، وجعلناهم ذكرى الدار يتذكر بأحوالهم المتذكر، ويعتبر بهم المعتبر، ويذكرون بأحسن الذكر.
{ وَإِنَّهُمْ عِنْدَنَا لَمِنَ الْمُصْطَفَيْنَ } الذين اصطفاهم اللّه من صفوة خلقه، { الأخْيَارِ } الذين لهم كل خلق كريم، وعمل مستقيم.
