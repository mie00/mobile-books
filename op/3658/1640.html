{ 41 } { إِنَّا أَنْزَلْنَا عَلَيْكَ الْكِتَابَ لِلنَّاسِ بِالْحَقِّ فَمَنِ اهْتَدَى فَلِنَفْسِهِ وَمَنْ ضَلَّ فَإِنَّمَا يَضِلُّ عَلَيْهَا وَمَا أَنْتَ عَلَيْهِمْ بِوَكِيلٍ } .
يخبر تعالى أنه أنزل على رسوله الكتاب المشتمل على الحق، في أخباره وأوامره ونواهيه، الذي هو مادة الهداية، وبلاغ لمن أراد الوصول إلى اللّه وإلى دار كرامته، وأنه قامت به الحجة على العالمين.
{ فَمَنِ اهْتَدَى } بنوره واتبع أوامره فإن نفع ذلك يعود إلى نفسه { وَمَنْ ضَلَّ } بعدما تبين له الهدى { فَإِنَّمَا يَضِلُّ عَلَيْهَا } لا يضر اللّه شيئا. { وَمَا أَنْتَ عَلَيْهِمْ بِوَكِيلٍ } تحفظ عليهم أعمالهم وتحاسبهم عليها، وتجبرهم على ما تشاء، وإنما أنت مبلغ تؤدي إليهم ما أمرت به.
