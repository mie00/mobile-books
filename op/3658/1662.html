{ الْيَوْمَ تُجْزَى كُلُّ نَفْسٍ بِمَا كَسَبَتْ لا ظُلْمَ الْيَوْمَ إِنَّ اللَّهَ سَرِيعُ الْحِسَابِ } .
{ الْيَوْمَ تُجْزَى كُلُّ نَفْسٍ بِمَا كَسَبَتْ } في الدنيا، من خير وشر، قليل وكثير. { لا ظُلْمَ الْيَوْمَ } على أحد، بزيادة في سيئاته، أو نقص من حسناته. { إِنَّ اللَّهَ سَرِيعُ الْحِسَابِ } أي: لا تستبطئوا ذلك اليوم فإنه آت، وكل آت قريب. وهو أيضا سريع المحاسبة لعباده يوم القيامة، لإحاطة علمه وكمال قدرته.
