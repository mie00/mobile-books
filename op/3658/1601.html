{ 164 - 166 } { وَمَا مِنَّا إِلا لَهُ مَقَامٌ مَعْلُومٌ * وَإِنَّا لَنَحْنُ الصَّافُّونَ * وَإِنَّا لَنَحْنُ الْمُسَبِّحُونَ } .
هذا [فيه] بيان براءة الملائكة عليهم السلام، عما قاله فيهم المشركون، وأنهم عباد اللّه، لا يعصونه طرفة عين، فما منهم من أحد إلا له مقام وتدبير قد أمره اللّه به لا يتعداه ولا يتجاوزه، وليس لهم من الأمر شيء.
{ وَإِنَّا لَنَحْنُ الصَّافُّونَ } في طاعة اللّه وخدمته.
{ وَإِنَّا لَنَحْنُ الْمُسَبِّحُونَ } لله عما لا يليق به. فكيف - مع هذا - يصلحون أن يكونوا شركاء للّه؟! تعالى الله.
