{ 7 } { وَالَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَنُكَفِّرَنَّ عَنْهُمْ سَيِّئَاتِهِمْ وَلَنَجْزِيَنَّهُمْ أَحْسَنَ الَّذِي كَانُوا يَعْمَلُونَ } .
يعني أن الذين منَّ اللّه عليهم بالإيمان والعمل الصالح، سيكفر اللّه عنهم سيئاتهم، لأن الحسنات يذهبن السيئات، { وَلَنَجْزِيَنَّهُمْ أَحْسَنَ الَّذِي كَانُوا يَعْمَلُونَ } [ ص 627 ] وهي أعمال الخير، من واجبات ومستحبات، فهي أحسن ما يعمل العبد، لأنه يعمل المباحات أيضا، وغيرها.
