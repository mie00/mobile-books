{ 12 } { وَجَعَلْنَا اللَّيْلَ وَالنَّهَارَ آيَتَيْنِ فَمَحَوْنَا آيَةَ اللَّيْلِ وَجَعَلْنَا آيَةَ النَّهَارِ مُبْصِرَةً لِتَبْتَغُوا فَضْلا مِنْ رَبِّكُمْ وَلِتَعْلَمُوا عَدَدَ السِّنِينَ وَالْحِسَابَ وَكُلَّ شَيْءٍ فَصَّلْنَاهُ تَفْصِيلا } .
يقول تعالى: { وَجَعَلْنَا اللَّيْلَ وَالنَّهَارَ آيَتَيْنِ } أي: دالتين على كمال قدرة الله وسعة رحمته وأنه الذي لا تنبغي العبادة إلا له. { فَمَحَوْنَا آيَةَ اللَّيْلِ } أي: جعلناه مظلما للسكون فيه والراحة، { وَجَعَلْنَا آيَةَ النَّهَارِ مُبْصِرَةً } أي: مضيئة { لِتَبْتَغُوا فَضْلا مِنْ رَبِّكُمْ } في معايشكم وصنائعكم وتجاراتكم وأسفاركم.
{ وَلِتَعْلَمُوا } بتوالي الليل والنهار واختلاف القمر { عَدَدَ السِّنِينَ وَالْحِسَابَ } فتبنون عليها ما تشاءون من مصالحكم.
