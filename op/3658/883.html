{ 35 } { مَثَلُ الْجَنَّةِ الَّتِي وُعِدَ الْمُتَّقُونَ تَجْرِي مِنْ تَحْتِهَا الأنْهَارُ أُكُلُهَا دَائِمٌ وَظِلُّهَا تِلْكَ عُقْبَى الَّذِينَ اتَّقَوْا وَعُقْبَى الْكَافِرِينَ النَّارُ } .
يقول تعالى: { مَثَلُ الْجَنَّةِ الَّتِي وُعِدَ الْمُتَّقُونَ } الذين تركوا ما نهاهم الله عنه، ولم يقصروا فيما أمرهم به، أي: صفتها وحقيقتها { تَجْرِي مِنْ تَحْتِهَا الأنْهَارُ } أنهار العسل، وأنهار الخمر، وأنهار اللبن، وأنهار الماء التي تجري في غير أخدود، فتسقى تلك البساتين والأشجار فتحمل من جميع أنواع الثمار.
{ أُكُلُهَا دَائِمٌ وَظِلُّهَا } دائم أيضا، { تِلْكَ عُقْبَى الَّذِينَ اتَّقَوْا } أي: عاقبتهم ومآلهم التي إليها يصيرون، { وَعُقْبَى الْكَافِرِينَ النَّارُ } فكم بين الفريقين من الفرق المبين؟!!
