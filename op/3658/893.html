{ وَإِذْ قَالَ مُوسَى لِقَوْمِهِ اذْكُرُوا نِعْمَةَ اللَّهِ عَلَيْكُمْ إِذْ أَنْجَاكُمْ مِنْ آلِ فِرْعَوْنَ يَسُومُونَكُمْ سُوءَ الْعَذَابِ وَيُذَبِّحُونَ أَبْنَاءَكُمْ وَيَسْتَحْيُونَ نِسَاءَكُمْ وَفِي ذَلِكُمْ بَلاءٌ مِنْ رَبِّكُمْ عَظِيمٌ * وَإِذْ تَأَذَّنَ رَبُّكُمْ لَئِنْ شَكَرْتُمْ لأزِيدَنَّكُمْ وَلَئِنْ كَفَرْتُمْ إِنَّ عَذَابِي لَشَدِيدٌ * وَقَالَ مُوسَى إِنْ تَكْفُرُوا أَنْتُمْ وَمَنْ فِي الأرْضِ جَمِيعًا فَإِنَّ اللَّهَ لَغَنِيٌّ حَمِيدٌ } .
فإنه يستدل بأيامه على كمال قدرته وعميم إحسانه، وتمام عدله وحكمته، ولهذا امتثل موسى عليه السلام أمر ربه، فذكرهم نعم الله فقال: { اذْكُرُوا نِعْمَةَ اللَّهِ عَلَيْكُمْ } أي: بقلوبكم وألسنتكم. { إِذْ أَنْجَاكُمْ مِنْ آلِ فِرْعَوْنَ يَسُومُونَكُمْ } أي: يولونكم { سُوءَ الْعَذَابِ } أي: أشده وفسر ذلك بقوله: { وَيُذَبِّحُونَ أَبْنَاءَكُمْ وَيَسْتَحْيُونَ نِسَاءَكُمْ } أي: يبقونهن فلا يقتلونهن، { وَفِي ذَلِكُمْ } الإنجاء { بَلاءٌ مِنْ رَبِّكُمْ عَظِيمٌ } أي: نعمة عظيمة، أو وفي ذلكم العذاب الذي ابتليتم به من فرعون وملئه ابتلاء من الله عظيم لكم، لينظر هل تصبرون أم لا؟
وقال لهم حاثا على شكر نعم الله: { وَإِذْ تَأَذَّنَ رَبُّكُمْ } أي: أعلم ووعد، { لَئِنْ شَكَرْتُمْ لأزِيدَنَّكُمْ } من نعمي { وَلَئِنْ كَفَرْتُمْ إِنَّ عَذَابِي لَشَدِيدٌ } ومن ذلك أن يزيل عنهم النعمة التي أنعم بها عليهم.
والشكر: هو اعتراف القلب بنعم الله والثناء على الله بها وصرفها في مرضاة الله تعالى. وكفر النعمة ضد ذلك.
{ وَقَالَ مُوسَى إِنْ تَكْفُرُوا أَنْتُمْ وَمَنْ فِي الأرْضِ جَمِيعًا } فلن تضروا الله شيئا، { فَإِنَّ اللَّهَ لَغَنِيٌّ حَمِيدٌ } فالطاعات لا تزيد في ملكه والمعاصي لا تنقصه، وهو كامل الغنى حميد في ذاته وأسمائه وصفاته وأفعاله، ليس له من الصفات إلا كل صفة حمد وكمال، ولا من الأسماء إلا كل اسم حسن، ولا من الأفعال إلا كل فعل جميل.
