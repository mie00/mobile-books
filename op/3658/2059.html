{ 41 - 45 } { إِنَّ الْمُتَّقِينَ فِي ظِلالٍ وَعُيُونٍ * وَفَوَاكِهَ مِمَّا يَشْتَهُونَ * كُلُوا وَاشْرَبُوا هَنِيئًا بِمَا كُنْتُمْ تَعْمَلُونَ * إِنَّا كَذَلِكَ نَجْزِي الْمُحْسِنِينَ * وَيْلٌ يَوْمَئِذٍ لِلْمُكَذِّبِينَ } .
لما ذكر عقوبة المكذبين، ذكر ثواب (1) المحسنين، فقال: { إِنَّ الْمُتَّقِينَ } [أي:] للتكذيب، المتصفين بالتصديق في أقوالهم وأفعالهم وأعمالهم، ولا يكونون كذلك إلا بأدائهم الواجبات، وتركهم المحرمات.
{ فِي ظِلالٍ } من كثرة الأشجار المتنوعة، الزاهية البهية. { وَعُيُونٍ } جارية من السلسبيل، والرحيق وغيرهما، { وَفَوَاكِهَ مِمَّا يَشْتَهُونَ } أي: من خيار الفواكه وطيبها، ويقال لهم: { كُلُوا وَاشْرَبُوا } من المآكل الشهية، والأشربة اللذيذة { هَنِيئًا } أي: من غير منغص ولا مكدر، ولا يتم هناؤه حتى يسلم الطعام والشراب من كل آفة ونقص، وحتى يجزموا أنه غير منقطع ولا زائل، { بِمَا كُنْتُمْ تَعْمَلُونَ } فأعمالكم هي السبب الموصل لكم إلى هذا النعيم (2) المقيم، وهكذا كل من أحسن في عبادة الله وأحسن إلى عباد الله، ولهذا قال: { إِنَّا كَذَلِكَ نَجْزِي الْمُحْسِنِينَ وَيْلٌ يَوْمَئِذٍ لِلْمُكَذِّبِينَ } ولو لم يكن لهم من هذا الويل إلا فوات هذا النعيم، لكفى به حرمانا وخسرانا (3) .
__________
(1) في ب: ثواب.
(2) في ب: إلى جنات النعيم.
(3) في ب: حزنا وحرمانا.
