{ يُبَصَّرُونَهُمْ يَوَدُّ الْمُجْرِمُ لَوْ يَفْتَدِي مِنْ عَذَابِ يَوْمِئِذٍ بِبَنِيهِ * وَصَاحِبَتِهِ وَأَخِيهِ * وَفَصِيلَتِهِ الَّتِي تُؤْوِيهِ * وَمَنْ فِي الأرْضِ جَمِيعًا ثُمَّ يُنْجِيهِ * كَلا إِنَّهَا لَظَى * نَزَّاعَةً لِلشَّوَى * تَدْعُوا مَنْ أَدْبَرَ وَتَوَلَّى * وَجَمَعَ فَأَوْعَى }
{ يَوَدُّ الْمُجْرِمُ } الذي حق عليه العذاب { لَوْ يَفْتَدِي مِنْ عَذَابِ يَوْمِئِذٍ بِبَنِيهِ وَصَاحِبَتِهِ } أي: زوجته { وَأَخِيهِ وَفَصِيلَتِهِ } أي: قرابته { الَّتِي تُؤْوِيهِ } أي: التي جرت عادتها في الدنيا أن تتناصر ويعين بعضها بعضا، ففي يوم القيامة، لا ينفع أحد أحدا، ولا يشفع أحد إلا بإذن الله.
بل لو يفتدي [المجرم المستحق للعذاب] بجميع ما في الأرضِ ثم ينجيه لم ينفعه ذلك.
{ كلا } أي: لا حيلة ولا مناص لهم، قد حقت عليهم كلمة ربك على الذين فسقوا أنهم لا يؤمنون (1) ، وذهب نفع الأقارب والأصدقاء.
[ ص 887 ] { إِنَّهَا لَظَى نزاعَةً لِلشَّوَى } أي: للأعضاء الظاهرة والباطنة من شدة عذابها (2) .
{ تَدْعُوا } إليها (3) { مَنْ أَدْبَرَ وَتَوَلَّى وجمع فأوعى } أي: أدبر عن اتباع الحق وأعرض عنه، فليس له فيه غرض، وجمع الأموال بعضها فوق بعض وأوعاها، فلم ينفق منها، فإن النار تدعوهم إلى نفسها، وتستعد للالتهاب بهم.
__________
(1) في ب: قد حقت عليهم كلمة ربك.
(2) في ب: أي: النار التي تتلظى تنزع من شدتها للأعضاء الظاهرة والباطنة.
(3) في ب: تدعو إلى نفسها.
