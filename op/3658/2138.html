{ 1 - 4 } { بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ قُلْ هُوَ اللَّهُ أَحَدٌ * اللَّهُ الصَّمَدُ * لَمْ يَلِدْ وَلَمْ يُولَدْ * وَلَمْ يَكُنْ لَهُ كُفُوًا أَحَدٌ } .
أي { قُلْ } قولا جازمًا به، معتقدًا له، عارفًا بمعناه، { هُوَ اللَّهُ أَحَدٌ } أي: قد انحصرت فيه الأحدية، فهو الأحد المنفرد بالكمال، الذي له الأسماء الحسنى، والصفات الكاملة العليا، والأفعال المقدسة، الذي لا نظير له ولا مثيل.
{ اللَّهُ الصَّمَدُ } أي: المقصود في جميع الحوائج. فأهل العالم العلوي والسفلي مفتقرون إليه غاية الافتقار، يسألونه حوائجهم، ويرغبون إليه في مهماتهم، لأنه الكامل في أوصافه، العليم الذي قد كمل في علمه، الحليم الذي قد كمل في حلمه، الرحيم الذي [كمل في رحمته الذي] وسعت رحمته كل شيء، وهكذا سائر أوصافه، ومن كماله أنه { لَمْ يَلِدْ وَلَمْ يُولَدْ } لكمال غناه { وَلَمْ يَكُنْ لَهُ كُفُوًا أَحَدٌ } لا في أسمائه ولا في أوصافه، ولا في أفعاله، تبارك وتعالى.
فهذه السورة مشتملة على توحيد الأسماء والصفات.
تفسير سورة الفلق
[وهي] مكية
