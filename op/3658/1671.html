{ 51 - 52 } { إِنَّا لَنَنْصُرُ رُسُلَنَا وَالَّذِينَ آمَنُوا فِي الْحَيَاةِ الدُّنْيَا وَيَوْمَ يَقُومُ الأشْهَادُ * يَوْمَ لا يَنْفَعُ الظَّالِمِينَ مَعْذِرَتُهُمْ وَلَهُمُ اللَّعْنَةُ وَلَهُمْ سُوءُ الدَّارِ } .
لما ذكر عقوبة آل فرعون في الدنيا، والبرزخ، ويوم القيامة، وذكر حالة أهل النار الفظيعة، الذين نابذوا رسله وحاربوهم، قال: { إِنَّا لَنَنْصُرُ رُسُلَنَا وَالَّذِينَ آمَنُوا فِي الْحَيَاةِ الدُّنْيَا } أي: بالحجة والبرهان والنصر، في الآخرة بالحكم لهم ولأتباعهم بالثواب، ولمن حاربهم بشدة العقاب.
{ يَوْمَ لا يَنْفَعُ الظَّالِمِينَ مَعْذِرَتُهُمْ } حين يعتذرون { وَلَهُمُ اللَّعْنَةُ وَلَهُمْ سُوءُ الدَّارِ } أي: الدار السيئة التي تسوء نازليها.
