{ 105 } { وَبِالْحَقِّ أَنْزَلْنَاهُ وَبِالْحَقِّ نَزَلَ وَمَا أَرْسَلْنَاكَ إِلا مُبَشِّرًا وَنَذِيرًا } .
أي: وبالحق أنزلنا هذا القرآن الكريم، لأمر العباد ونهيهم، وثوابهم وعقابهم، { وَبِالْحَقِّ نزلَ } أي: بالصدق والعدل والحفظ من كل شيطان رجيم { وَمَا أَرْسَلْنَاكَ إِلا مُبَشِّرًا } من أطاع الله بالثواب العاجل والآجل { وَنَذِيرًا } لمن عصى الله بالعقاب العاجل والآجل، ويلزم من ذلك بيان ما بشر به وأنذر.
