{ 40 - 42 } { وَيَوْمَ يَحْشُرُهُمْ جَمِيعًا ثُمَّ يَقُولُ لِلْمَلائِكَةِ أَهَؤُلاءِ إِيَّاكُمْ كَانُوا يَعْبُدُونَ * قَالُوا سُبْحَانَكَ أَنْتَ وَلِيُّنَا مِنْ دُونِهِمْ بَلْ كَانُوا يَعْبُدُونَ الْجِنَّ أَكْثَرُهُمْ بِهِمْ مُؤْمِنُونَ * فَالْيَوْمَ لا يَمْلِكُ بَعْضُكُمْ لِبَعْضٍ نَفْعًا وَلا ضَرًّا وَنَقُولُ لِلَّذِينَ ظَلَمُوا ذُوقُوا عَذَابَ النَّارِ الَّتِي كُنْتُمْ بِهَا تُكَذِّبُونَ } .
{ وَيَوْمَ يَحْشُرُهُمْ جَمِيعًا } أي: العابدين لغير اللّه [ ص 682 ] والمعبودين من دونه، من الملائكة. { ثُمَّ يَقُولُ } الله { لِلْمَلائِكَةِ } على وجه التوبيخ لمن عبدهم { أَهَؤُلاءِ إِيَّاكُمْ كَانُوا يَعْبُدُونَ } فتبرأوا من عبادتهم.
و { قَالُوا سُبْحَانَكَ } أي: تنزيها لك وتقديسا، أن يكون لك شريك، أو ند { أَنْتَ وَلِيُّنَا مِنْ دُونِهِمْ } فنحن مفتقرون إلى ولايتك، مضطرون إليها، فكيف ندعو غيرنا إلى عبادتنا؟ أم كيف نصلح لأن نتخذ من دونك أولياء وشركاء؟"
ولكن هؤلاء المشركون { كَانُوا يَعْبُدُونَ الْجِنَّ } أي: الشياطين، يأمرون (1) بعبادتنا أو عبادة غيرنا، فيطيعونهم بذلك. وطاعتهم هي عبادتهم، لأن العبادة الطاعة، كما قال تعالى مخاطبا لكل من اتخذ معه آلهة { أَلَمْ أَعْهَدْ إِلَيْكُمْ يَا بَنِي آدَمَ أَنْ لا تَعْبُدُوا الشَّيْطَانَ إِنَّهُ لَكُمْ عَدُوٌّ مُبِينٌ * وَأَنِ اعْبُدُونِي هَذَا صِرَاطٌ مُسْتَقِيمٌ }
{ أَكْثَرُهُمْ بِهِمْ مُؤْمِنُونَ } أي مصدقون للجن منقادون لهم لأن الإيمان هو التصديق الموجب للانقياد
فلما تبرأوا منهم قال تعالى مخاطبا لهم { فَالْيَوْمَ لا يَمْلِكُ بَعْضُكُمْ لِبَعْضٍ نَفْعًا وَلا ضَرًّا } تقطعت بينكم الأسباب وانقطع بعضكم من بعض { وَنَقُولُ لِلَّذِينَ ظَلَمُوا } بالكفر والمعاصي - بعد ما ندخلهم النار - { ذُوقُوا عَذَابَ النَّارِ الَّتِي كُنْتُمْ بِهَا تُكَذِّبُونَ } فاليوم عاينتموها ودخلتموها جزاء لتكذيبكم وعقوبة لما أحدثه ذلك التكذيب من عدم الهرب من أسبابها
__________
(1) في ب: يأمرونهم.
