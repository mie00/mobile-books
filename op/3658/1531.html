{ قُلْ } لهم - مخبرا بوقت وقوعه الذي لا شك فيه -: { لَكُمْ مِيعَادُ يَوْمٍ لا تَسْتَأْخِرُونَ عَنْهُ سَاعَةً وَلا تَسْتَقْدِمُونَ } فاحذروا ذلك اليوم، وأعدوا له عدته.
[ ص 681 ]
{ 31 - 33 } { وَقَالَ الَّذِينَ كَفَرُوا لَنْ نُؤْمِنَ بِهَذَا الْقُرْآنِ وَلا بِالَّذِي بَيْنَ يَدَيْهِ وَلَوْ تَرَى إِذِ الظَّالِمُونَ مَوْقُوفُونَ عِنْدَ رَبِّهِمْ يَرْجِعُ بَعْضُهُمْ إِلَى بَعْضٍ الْقَوْلَ يَقُولُ الَّذِينَ اسْتُضْعِفُوا لِلَّذِينَ اسْتَكْبَرُوا لَوْلا أَنْتُمْ لَكُنَّا مُؤْمِنِينَ } .
لما ذكر تعالى أن ميعاد المستعجلين بالعذاب، لا بد من وقوعه عند حلول أجله، ذكر هنا حالهم في ذلك اليوم، وأنك لو رأيت حالهم إذا وقفوا عند ربهم، واجتمع الرؤساء والأتباع في الكفر والضلال، لرأيت أمرا عظيما وهولا جسيما، ورأيت كيف يتراجع، ويرجع بعضهم إلى بعض القول، فـ { يَقُولُ الَّذِينَ اسْتُضْعِفُوا } وهم الأتباع { لِلَّذِينَ اسْتَكْبَرُوا } وهم القادة: { لَوْلا أَنْتُمْ لَكُنَّا مُؤْمِنِينَ } ولكنكم حُلْتُم بيننا وبين الإيمان، وزينتم لنا الكفر[ان]، فتبعناكم على ذلك، ومقصودهم بذلك أن يكون العذاب على الرؤساء دونهم.
