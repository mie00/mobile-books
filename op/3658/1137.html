{ 47 - 48 } { فَأْتِيَاهُ فَقُولا إِنَّا رَسُولا رَبِّكَ فَأَرْسِلْ مَعَنَا بَنِي إِسْرَائِيلَ وَلا تُعَذِّبْهُمْ قَدْ جِئْنَاكَ بِآيَةٍ مِنْ رَبِّكَ وَالسَّلامُ عَلَى مَنِ اتَّبَعَ الْهُدَى * إِنَّا قَدْ أُوحِيَ إِلَيْنَا أَنَّ الْعَذَابَ عَلَى مَنْ كَذَّبَ وَتَوَلَّى } .
أي: فأتياه بهذين الأمرين، دعوته إلى الإسلام، وتخليص هذا الشعب الشريف بني إسرائيل -من قيده وتعبيده لهم، ليتحرروا ويملكوا أمرهم، ويقيم فيهم موسى شرع الله ودينه.
{ قَدْ جِئْنَاكَ بِآيَةٍ } تدل على صدقنا { فَأَلْقَى } موسى { عَصَاهُ فَإِذَا هِيَ ثُعْبَانٌ مُبِينٌ * وَنزعَ يَدَهُ فَإِذَا هِيَ بَيْضَاءُ لِلنَّاظِرِينَ } إلى آخر ما ذكر الله عنهما
{ وَالسَّلامُ عَلَى مَنِ اتَّبَعَ الْهُدَى } أي من اتبع الصراط المستقيم واهتدى بالشرع المبين حصلت له السلامة في الدنيا والآخرة
