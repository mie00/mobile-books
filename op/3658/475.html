أي: { وَلَوْ تَرَى } الكافرين { إِذْ وُقِفُوا عَلَى رَبِّهِمْ } لرأيت أمرا عظيما، وهَوْلا جسيما، { قَالَ } لهم موبخا ومقرعا: { أَلَيْسَ هَذَا } الذي ترون من العذاب { بِالْحَقِّ قَالُوا بَلَى وَرَبِّنَا } فأقروا، واعترفوا حيث لا ينفعهم ذلك، { قَالَ فَذُوقُوا الْعَذَابَ بِمَا كُنْتُمْ تَكْفُرُونَ } . { 31 } { قَدْ خَسِرَ الَّذِينَ كَذَّبُوا بِلِقَاءِ اللَّهِ حَتَّى إِذَا جَاءَتْهُمُ السَّاعَةُ بَغْتَةً قَالُوا يَا حَسْرَتَنَا عَلَى مَا فَرَّطْنَا فِيهَا وَهُمْ يَحْمِلُونَ أَوْزَارَهُمْ عَلَى ظُهُورِهِمْ أَلا سَاءَ مَا يَزِرُونَ } .
