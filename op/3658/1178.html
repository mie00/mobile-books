{ 34 - 35 } { وَمَا جَعَلْنَا لِبَشَرٍ مِنْ قَبْلِكَ الْخُلْدَ أَفَإِنْ مِتَّ فَهُمُ الْخَالِدُونَ * كُلُّ نَفْسٍ ذَائِقَةُ الْمَوْتِ وَنَبْلُوكُمْ بِالشَّرِّ وَالْخَيْرِ فِتْنَةً وَإِلَيْنَا تُرْجَعُونَ } .
لما كان أعداء الرسول يقولون (1) تربصوا به ريب المنون. قال الله تعالى: هذا طريق مسلوك، ومعبد منهوك، فلم نجعل لبشر { مِنْ قَبْلِكَ } يا محمد { الْخُلْدَ } في الدنيا، فإذا مت، فسبيل أمثالك، من الرسل والأنبياء، والأولياء، وغيرهم.
{ أَفَإِنْ مِتَّ فَهُمُ الْخَالِدُونَ } أي: فهل إذا مت خلدوا بعدك، فليهنهم الخلود إذًا إن كان، وليس الأمر كذلك، بل كل من عليها فان، ولهذا قال: { كُلُّ نَفْسٍ ذَائِقَةُ الْمَوْتِ } وهذا يشمل سائر نفوس الخلائق، وإن هذا كأس لا بد من شربه وإن طال بالعبد المدى، وعمّر سنين، ولكن الله تعالى أوجد عباده في الدنيا، وأمرهم، ونهاهم، وابتلاهم بالخير والشر، بالغنى والفقر، والعز والذل والحياة والموت، فتنة منه تعالى ليبلوهم أيهم أحسن عملا ومن يفتتن عند مواقع الفتن ومن ينجو، { وَإِلَيْنَا تُرْجَعُونَ } فنجازيكم بأعمالكم، إن خيرا فخير، وإن شرا فشر { وَمَا رَبُّكَ بِظَلامٍ لِلْعَبِيدِ } وهذه الآية، تدل على بطلان قول من يقول ببقاء الخضر، وأنه مخلد في الدنيا، فهو قول، لا دليل عليه، ومناقض للأدلة الشرعية.
__________
(1) في النسختين: يقولون قل تربصوا.
