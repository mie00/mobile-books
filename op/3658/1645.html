{ وَبَدَا لَهُمْ سَيِّئَاتُ مَا كَسَبُوا وَحَاقَ بِهِمْ مَا كَانُوا بِهِ يَسْتَهْزِئُونَ } .
{ وَبَدَا لَهُمْ سَيِّئَاتُ مَا كَسَبُوا } أي: الأمور التي تسوؤهم، بسبب صنيعهم وكسبهم. { وَحَاقَ بِهِمْ مَا كَانُوا بِهِ يَسْتَهْزِئُونَ } من الوعيد والعذاب الذي نزل بهم، وما حل عليهم العقاب.
