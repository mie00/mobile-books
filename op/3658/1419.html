أي { و } أرسلنا { إِلَى مَدْيَنَ } القبيلة المعروفة المشهورة { شُعَيْبًا } فأمرهم بعبادة اللّه وحده لا شريك له، والإيمان بالبعث ورجائه، والعمل له، ونهاهم عن الإفساد في الأرض، ببخس المكاييل والموازين، والسعي بقطع الطرق، فكذبوه فأخذهم عذاب اللّه { فَأَصْبَحُوا فِي دَارِهِمْ جَاثِمِينَ } { 38-40 } { وَعَادًا وَثَمُودَ وَقَدْ تَبَيَّنَ لَكُمْ مِنْ مَسَاكِنِهِمْ وَزَيَّنَ لَهُمُ الشَّيْطَانُ أَعْمَالَهُمْ فَصَدَّهُمْ عَنِ السَّبِيلِ وَكَانُوا مُسْتَبْصِرِينَ } .
أي: وكذلك ما فعلنا بعاد وثمود، وقد علمتم قصصهم، وتبين لكم بشيء [ ص 631 ] تشاهدونه بأبصاركم من مساكنهم وآثارهم التي بانوا عنها، وقد جاءتهم رسلهم بالآيات البينات، المفيدة للبصيرة، فكذبوهم وجادلوهم. { وَزَيَّنَ لَهُمُ الشَّيْطَانُ أَعْمَالَهُمْ } حتى ظنوا أنها أفضل مما جاءتهم به الرسل.
