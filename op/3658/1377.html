{ 87 - 90 } { وَيَوْمَ يُنْفَخُ فِي الصُّورِ فَفَزِعَ مَنْ فِي السَّمَاوَاتِ وَمَنْ فِي الأرْضِ إِلا مَنْ شَاءَ اللَّهُ وَكُلٌّ أَتَوْهُ دَاخِرِينَ * وَتَرَى الْجِبَالَ تَحْسَبُهَا جَامِدَةً وَهِيَ تَمُرُّ مَرَّ السَّحَابِ صُنْعَ اللَّهِ الَّذِي أَتْقَنَ كُلَّ شَيْءٍ إِنَّهُ خَبِيرٌ بِمَا تَفْعَلُونَ } .
يخوف تعالى عباده ما أمامهم من يوم القيامة وما فيه من المحن والكروب، ومزعجات القلوب فقال: { وَيَوْمَ يُنْفَخُ فِي الصُّورِ فَفَزِعَ } بسبب النفخ فيه { مَنْ فِي السَّمَاوَاتِ وَمَنْ فِي الأرْضِ } أي: انزعجوا وارتاعوا وماج بعضهم ببعض خوفا مما هو مقدمة له. { إِلا مَنْ شَاءَ اللَّهُ } ممن أكرمه الله وثبته وحفظه من الفزع. { وَكُلٌّ } من الخلق عند النفخ في الصور { أَتَوْهُ دَاخِرِينَ } صاغرين ذليلين، كما قال تعالى: { إِنْ كُلُّ مَنْ فِي السَّمَاوَاتِ وَالأرْضِ إِلا آتِي الرَّحْمَنِ عَبْدًا } ففي ذلك اليوم يتساوى الرؤساء والمرءوسون في الذل والخضوع لمالك الملك.
ومن هوله أنك { ترى الْجِبَالَ تَحْسَبُهَا جَامِدَةً } لا تفقد [شيئا] منها وتظنها باقية على الحال المعهودة وهي قد بلغت منها الشدائد والأهوال كل مبلغ وقد تفتت ثم تضمحل وتكون هباء منبثا. ولهذا قال: { وَهِيَ تَمُرُّ مَرَّ السَّحَابِ } من خفتها وشدة ذلك الخوف وذلك { صُنْعَ اللَّهِ الَّذِي أَتْقَنَ كُلَّ شَيْءٍ إِنَّهُ خَبِيرٌ بِمَا تَفْعَلُونَ } فيجازيكم بأعمالكم.
