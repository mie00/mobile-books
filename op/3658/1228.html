{ الْمُلْكُ يَوْمَئِذٍ لِلَّهِ يَحْكُمُ بَيْنَهُمْ فَالَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ فِي جَنَّاتِ النَّعِيمِ * وَالَّذِينَ كَفَرُوا وَكَذَّبُوا بِآيَاتِنَا فَأُولَئِكَ لَهُمْ عَذَابٌ مُهِينٌ } .
{ الْمُلْكُ يَوْمَئِذٍ } أي: يوم القيامة { لِلَّهِ } تعالى، لا لغيره، { يَحْكُمُ بَيْنَهُمْ } بحكمه العدل، وقضائه الفصل، { فَالَّذِينَ آمَنُوا } بالله ورسله، وما جاءوا به { وَعَمِلُوا الصَّالِحَاتِ } ليصدقوا بذلك إيمانهم { فِي جَنَّاتِ النَّعِيمِ } نعيم القلب والروح والبدن، مما لا يصفه الواصفون، ولا تدركه العقول.
{ وَالَّذِينَ كَفَرُوا } بالله ورسله وكذبوا بآياته الهادية للحق والصواب فأعرضوا عنها، أو عاندوها، { فَأُولَئِكَ لَهُمْ عَذَابٌ مُهِينٌ } لهم، من شدته، وألمه، وبلوغه للأفئدة كما استهانوا برسله وآياته، أهانهم الله بالعذاب.
