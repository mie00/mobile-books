{ فَلَيْسَ لَهُ الْيَوْمَ هَاهُنَا حَمِيمٌ * وَلا طَعَامٌ إِلا مِنْ غِسْلِينٍ * لا يَأْكُلُهُ إِلا الْخَاطِئُونَ } .
{ فَلَيْسَ لَهُ الْيَوْمَ هَاهُنَا } أي: يوم القيامة { حَمِيمٍ } أي: قريب أو صديق يشفع له لينجو من عذاب الله أو يفوز بثواب الله: { وَلا تَنْفَعُ الشَّفَاعَةُ عِنْدَهُ إِلا لِمَنْ أَذِنَ لَهُ } { مَا لِلظَّالِمِينَ مِنْ حَمِيمٍ وَلا شَفِيعٍ يُطَاعُ } .
وليس له طعام إلا من غسلين وهو صديد أهل النار، الذي هو في غاية الحرارة، ونتن الريح، وقبح الطعم ومرارته لا يأكل هذا الطعام الذميم { إِلا الْخَاطِئُونَ } الذين أخطأوا الصراط المستقيم وسلكوا سبل الجحيم (1) فلذلك استحقوا العذاب الأليم.
__________
(1) في ب: وسلكوا كل طريق يوصلهم إلى الجحيم.
