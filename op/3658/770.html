{ 84 } { وَقَالَ مُوسَى } موصيًا لقومه بالصبر، ومذكرًا لهم ما يستعينون به على ذلك فقال: { يَا قَوْمِ إِنْ كُنْتُمْ آمَنْتُمْ بِاللَّهِ } فقوموا بوظيفة [ ص 372 ] الإيمان.
{ فَعَلَيْهِ تَوَكَّلُوا إِنْ كُنْتُمْ مُسْلِمِينَ } أي: اعتمدوا عليه، والجؤوا إليه واستنصروه.
