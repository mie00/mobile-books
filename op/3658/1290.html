{ 46 } { لَقَدْ أَنْزَلْنَا آيَاتٍ مُبَيِّنَاتٍ وَاللَّهُ يَهْدِي مَنْ يَشَاءُ إِلَى صِرَاطٍ مُسْتَقِيمٍ } .
أي: لقد رحمنا عبادنا، وأنزلنا إليهم آيات بينات، أي: واضحات الدلالة، على جميع المقاصد الشرعية، والآداب المحمودة، والمعارف الرشيدة، فاتضحت بذلك السبل، وتبين الرشد من الغي، والهدى من الضلال، فلم يبق أدنى شبهة لمبطل يتعلق بها، ولا أدنى إشكال لمريد الصواب، لأنها تنزيل من كمل علمه، وكملت رحمته، وكمل بيانه، فليس بعد بيانه بيان { لِيَهْلِكَ } بعد ذلك { مَنْ هَلَكَ عَنْ بَيِّنَةٍ وَيَحْيَا مَنْ حَيَّ عَنْ بَيِّنَةٍ } { وَاللَّهُ يَهْدِي مَنْ يَشَاءُ } ممن سبقت لهم سابقة الحسنى، وقدم الصدق، { إِلَى صِرَاطٍ مُسْتَقِيمٍ } أي: طريق واضح مختصر، موصل إليه، وإلى دار كرامته، متضمن العلم بالحق وإيثاره والعمل به. عمم البيان التام لجميع الخلق، وخصص بالهداية من يشاء، فهذا فضله وإحسانه، وما فضل الكريم بممنون وذاك عدله، وقطع الحجة للمحتج، والله أعلم حيث يجعل مواقع إحسانه.
