{ 36 - 39 } { فَمَالِ الَّذِينَ كَفَرُوا قِبَلَكَ مُهْطِعِينَ * عَنِ الْيَمِينِ وَعَنِ الشِّمَالِ عِزِينَ * أَيَطْمَعُ كُلُّ امْرِئٍ مِنْهُمْ أَنْ يُدْخَلَ جَنَّةَ نَعِيمٍ * كَلا إِنَّا خَلَقْنَاهُمْ مِمَّا يَعْلَمُونَ } .
يقول تعالى، مبينا اغترار الكافرين: { فَمَالِ الَّذِينَ كَفَرُوا قِبَلَكَ مُهْطِعِينَ } أي: مسرعين.
{ عَنِ الْيَمِينِ وَعَنِ الشِّمَالِ عِزِينَ } أي: قطعا متفرقة وجماعات متوزعة (1) ، كل منهم بما لديه فرح.
{ أَيَطْمَعُ كُلُّ امْرِئٍ مِنْهُمْ أَنْ يُدْخَلَ جَنَّةَ نَعِيمٍ } بأي: سبب أطمعهم، وهم لم يقدموا سوى الكفر، والجحود برب العالمين، ولهذا قال: { كلا } [أي:] ليس الأمر بأمانيهم ولا إدراك ما يشتهون بقوتهم.
{ إِنَّا خَلَقْنَاهُمْ مِمَّا يَعْلَمُونَ } أي: من ماء دافق، يخرج من بين الصلب والترائب، فهم ضعفاء، لا يملكون لأنفسهم نفعا ولا ضرا، ولا موتا ولا حياة ولا نشورا.
__________
(1) في ب: متنوعة.
