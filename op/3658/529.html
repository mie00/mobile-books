{ 147 } { فَإِنْ كَذَّبُوكَ فَقُلْ رَبُّكُمْ ذُو رَحْمَةٍ وَاسِعَةٍ وَلا يُرَدُّ بَأْسُهُ عَنِ الْقَوْمِ الْمُجْرِمِينَ } .
أي: فإن كذبك هؤلاء المشركون، فاستمر على دعوتهم، بالترغيب والترهيب،وأخبرهم بأن الله { ذُو رَحْمَةٍ وَاسِعَةٍ } أي: عامة شاملة [لجميع] المخلوقات كلها، فسارعوا إلى رحمته بأسبابها، التي رأسها وأسها ومادتها، تصديق محمد صلى الله عليه وسلم فيما جاء به.
{ وَلا يُرَدُّ بَأْسُهُ عَنِ الْقَوْمِ الْمُجْرِمِينَ } أي: الذين كثر إجرامهم وذنوبهم.فاحذروا الجرائم الموصلة لبأس الله، التي أعظمها ورأسها تكذيب محمد صلى الله عليه وسلم.
