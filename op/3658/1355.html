{ وَجَحَدُوا بِهَا } أي: كفروا بآيات الله جاحدين لها، { وَاسْتَيْقَنَتْهَا أَنْفُسُهُمْ } أي: ليس جحدهم مستندا إلى الشك والريب، وإنما جحدهم مع علمهم ويقينهم (1) بصحتها { ظُلْمًا } منهم لحق ربهم ولأنفسهم، { وَعُلُوًّا } على الحق وعلى العباد وعلى الانقياد للرسل، { فَانْظُرْ كَيْفَ كَانَ عَاقِبَةُ الْمُفْسِدِينَ } أسوأ عاقبة دمرهم الله وغرقهم في البحر وأخزاهم وأورث مساكنهم المستضعفين من عباده.
__________
(1) في ب: تيقنهم.
