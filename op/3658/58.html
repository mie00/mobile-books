{ 63-64 } { وَإِذْ أَخَذْنَا مِيثَاقَكُمْ وَرَفَعْنَا فَوْقَكُمُ الطُّورَ خُذُوا مَا آتَيْنَاكُمْ بِقُوَّةٍ وَاذْكُرُوا مَا فِيهِ لَعَلَّكُمْ تَتَّقُونَ * ثُمَّ تَوَلَّيْتُمْ مِنْ بَعْدِ ذَلِكَ فَلَوْلا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ لَكُنْتُمْ مِنَ الْخَاسِرِينَ } .
أي: واذكروا { إِذْ أَخَذْنَا مِيثَاقَكُمْ } وهو العهد الثقيل المؤكد بالتخويف لهم، برفع الطور فوقهم (1) وقيل لهم: { خُذُوا مَا آتَيْنَاكُمْ } من التوراة { بِقُوَّةٍ } أي: بجد واجتهاد، وصبر على أوامر الله، { وَاذْكُرُوا مَا فِيهِ } أي: ما في كتابكم بأن تتلوه وتتعلموه، { لَعَلَّكُمْ تَتَّقُونَ } عذاب الله وسخطه، أو لتكونوا من أهل التقوى.
فبعد هذا التأكيد البليغ { تَوَلَّيْتُمْ } وأعرضتم، وكان ذلك موجبا لأن يحل بكم أعظم العقوبات، ولكن { لَوْلا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ لَكُنْتُمْ مِنَ الْخَاسِرِينَ } .
__________
(1) كذا في ب، وفي أ: برفع الطور فوقكم.
