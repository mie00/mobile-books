{ 7 - 9 } { وَقَالَ الَّذِينَ كَفَرُوا هَلْ نَدُلُّكُمْ عَلَى رَجُلٍ يُنَبِّئُكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّقٍ إِنَّكُمْ لَفِي خَلْقٍ جَدِيدٍ } .
أي: { وَقَالَ الَّذِينَ كَفَرُوا } على وجه التكذيب والاستهزاء والاستبعاد، وذكر وجه الاستبعاد.
أي: قال بعضهم لبعض: { هَلْ نَدُلُّكُمْ عَلَى رَجُلٍ يُنَبِّئُكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّقٍ إِنَّكُمْ لَفِي خَلْقٍ جَدِيدٍ } يعنون بذلك الرجل، رسول اللّه صلى اللّه عليه وسلم، وأنه رجل أتى بما يستغرب منه، حتى صار - بزعمهم - فرجة يتفرجون عليه، وأعجوبة يسخرون منه، وأنه كيف يقول { إِنَّكُمْ مَبْعُوثُونَ } بعدما مزقكم البلى، وتفرقت أوصالكم، واضمحلت أعضاؤكم؟!.
