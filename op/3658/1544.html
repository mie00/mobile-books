{ 9 } { وَاللَّهُ الَّذِي أَرْسَلَ الرِّيَاحَ فَتُثِيرُ سَحَابًا فَسُقْنَاهُ إِلَى بَلَدٍ مَيِّتٍ فَأَحْيَيْنَا بِهِ الأرْضَ بَعْدَ مَوْتِهَا كَذَلِكَ النُّشُورُ } .
يخبر تعالى عن كمال اقتداره، وسعة جوده، وأنه { أَرْسَلَ الرِّيَاحَ فَتُثِيرُ سَحَابًا فَسُقْنَاهُ إِلَى بَلَدٍ مَيِّتٍ } فأنزله اللّه عليها { فَأَحْيَيْنَا بِهِ الأرْضَ بَعْدَ مَوْتِهَا }
فحييت البلاد والعباد، وارتزقت الحيوانات، ورتعت في تلك الخيرات، { كَذَلِكَ } الذي أحيا الأرض بعد موتها، ينشر الله الأموات من قبورهم، بعدما مزقهم البلى، فيسوق إليهم مطرا، كما ساقه إلى الأرض الميتة، فينزله عليهم فتحيا الأجساد والأرواح من القبور، ويأتون للقيام بين يدي الله ليحكم بينهم، ويفصل بحكمه العدل.
