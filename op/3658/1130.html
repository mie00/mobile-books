{ إِنَّ السَّاعَةَ آتِيَةٌ } أي: لا بد من وقوعها { أَكَادُ أُخْفِيهَا } أي: عن نفسي كما في بعض القراءات، كقوله تعالى: { يَسْأَلَكَ الناس عَنِ السَّاعَةِ قُلْ إِنَّمَا عِلْمُهَا عِنْدَ الله } وقال: { وَعِنْدَهُ عِلْمُ السَّاعَةِ } فعلمها قد أخفاه عن الخلائق كلهم، فلا يعلمها ملك مقرب، ولا نبي مرسل، والحكمة في إتيان الساعة { لِتُجْزَى كُلُّ نَفْسٍ بِمَا تَسْعَى } من الخير والشر، فهي الباب لدار الجزاء { لِيَجْزِيَ الَّذِينَ أَسَاءُوا بِمَا عَمِلُوا وَيَجْزِيَ الَّذِينَ أَحْسَنُوا بِالْحُسْنَى }
