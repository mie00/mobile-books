{ 64 - 65 } { وَمَا نَتَنَزَّلُ إِلا بِأَمْرِ رَبِّكَ لَهُ مَا بَيْنَ أَيْدِينَا وَمَا خَلْفَنَا وَمَا بَيْنَ ذَلِكَ وَمَا كَانَ رَبُّكَ نَسِيًّا } .
استبطأ النبي صلى الله عليه وسلم جبريل عليه السلام مرة في نزوله إليه فقال له: " لو تأتينا أكثر مما تأتينا " -تشوقا إليه، وتوحشا [ ص 498 ] لفراقه، وليطمئن قلبه بنزوله- فأنزل الله تعالى على لسان جبريل: { وَمَا نَتَنَزَّلُ إِلا بِأَمْرِ رَبِّكَ } أي: ليس لنا من الأمر شيء، إن أمرنا، ابتدرنا أمره، ولم نعص له أمرا، كما قال عنهم: { لا يَعْصُونَ اللَّهَ مَا أَمَرَهُمْ وَيَفْعَلُونَ مَا يُؤْمَرُونَ } فنحن عبيد مأمورون، { لَهُ مَا بَيْنَ أَيْدِينَا وَمَا خَلْفَنَا وَمَا بَيْنَ ذَلِكَ } أي: له الأمور الماضية والمستقبلة والحاضرة، في الزمان والمكان، فإذا تبين أن الأمر كله لله، وأننا عبيد مدبرون، فيبقى الأمر دائرا بين: " هل تقتضيه الحكمة الإلهية فينفذه؟ أم لا تقتضيه فيؤخره " ؟ ولهذا قال: { وَمَا كَانَ رَبُّكَ نَسِيًّا } أي: لم يكن لينساك ويهملك، كما قال تعالى: { مَا وَدَّعَكَ رَبُّكَ وَمَا قَلَى } بل لم يزل معتنيا بأمورك، مجريا لك على أحسن عوائده الجميلة، وتدابيره الجميلة.
أي: فإذا تأخر نزولنا عن الوقت المعتاد، فلا يحزنك ذلك ولا يهمك، واعلم أن الله هو الذي أراد ذلك، لما له من الحكمة فيه.
