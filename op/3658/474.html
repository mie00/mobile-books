{ وَلَوْ رُدُّوا لَعَادُوا لِمَا نُهُوا عَنْهُ وَإِنَّهُمْ لَكَاذِبُونَ * وَقَالُوا } منكرين للبعث { إِنْ هِيَ إِلا حَيَاتُنَا الدُّنْيَا } أي ما حقيقة الحال والأمر وما المقصود من إيجادنا إلا الحياة الدنيا وحدها { وَمَا نَحْنُ بِمَبْعُوثِينَ }.{ 30 } { وَلَوْ تَرَى إِذْ وُقِفُوا عَلَى رَبِّهِمْ قَالَ أَلَيْسَ هَذَا بِالْحَقِّ قَالُوا بَلَى وَرَبِّنَا قَالَ فَذُوقُوا الْعَذَابَ بِمَا كُنْتُمْ تَكْفُرُونَ } .
