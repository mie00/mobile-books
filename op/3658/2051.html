{ وَمِنَ اللَّيْلِ فَاسْجُدْ لَهُ } أي: أكثر [له] من السجود، ولا يكون ذلك إلا بالإكثار من الصلاة (1) . { وَسَبِّحْهُ لَيْلا طَوِيلا } وقد تقدم تقييد هذا المطلق بقوله: { يَا أَيُّهَا الْمُزَّمِّلُ قُمِ اللَّيْلَ إِلا قَلِيلا } الآية (2) [وقوله] { إِنَّ هَؤُلاءِ } أي: المكذبين لك أيها الرسول بعد ما بينت لهم الآيات، ورغبوا ورهبوا، ومع ذلك، لم يفد فيهم ذلك شيئا، بل لا يزالون يؤثرون، { الْعَاجِلَةَ } ويطمئنون إليها، { وَيَذَرُونَ } أي: يتركون العمل ويهملون { وَرَاءَهُمْ } أي: أمامهم { يَوْمًا ثَقِيلا } وهو يوم القيامة، الذي مقداره خمسون ألف سنة مما تعدون، وقال تعالى: { يَقُولُ الْكَافِرُونَ هَذَا يَوْمٌ عَسِرٌ } فكأنهم ما خلقوا إلا للدنيا والإقامة فيها.
__________
(1) في ب: وذلك متضمن لكثرة الصلاة.
(2) "في ب: أكمل الآيات (نِصْفَهُ أَوِ انْقُصْ مِنْهُ قَلِيلا أَوْ زِدْ عَلَيْهِ).
