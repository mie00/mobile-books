{ 18 } { فَهَلْ يَنْظُرُونَ إِلا السَّاعَةَ أَنْ تَأْتِيَهُمْ بَغْتَةً فَقَدْ جَاءَ أَشْرَاطُهَا فَأَنَّى لَهُمْ إِذَا جَاءَتْهُمْ ذِكْرَاهُمْ } .
أي: فهل ينظر هؤلاء المكذبون أو ينتظرون { إِلا السَّاعَةَ أَنْ تَأْتِيَهُمْ بَغْتَةً } أي: فجأة، وهم لا يشعرون { فَقَدْ جَاءَ أَشْرَاطُهَا } أي: علاماتها الدالة على قربها.
{ فَأَنَّى لَهُمْ إِذَا جَاءَتْهُمْ ذِكْرَاهُمْ } أي: من أين لهم، إذا جاءتهم الساعة وانقطعت آجالهم أن يتذكروا ويستعتبوا؟ قد فات ذلك، وذهب وقت التذكر، فقد عمروا ما يتذكر فيه من تذكر، وجاءهم النذير.
ففي هذا الحث على الاستعداد قبل مفاجأة الموت، فإن موت الإنسان قيام ساعته.
