{ 158 - 160 } { وَجَعَلُوا بَيْنَهُ وَبَيْنَ الْجِنَّةِ نَسَبًا وَلَقَدْ عَلِمَتِ الْجِنَّةُ إِنَّهُمْ لَمُحْضَرُونَ * سُبْحَانَ اللَّهِ عَمَّا يَصِفُونَ * إِلا عِبَادَ اللَّهِ الْمُخْلَصِينَ } .
أي: جعل هؤلاء المشركون باللّه بين اللّه وبين الجنة نسبا، حيث زعموا أن الملائكة بنات اللّه، وأن أمهاتهم سروات الجن، والحال أن الجنة قد علمت أنهم محضرون بين يدي اللّه، [ليجازيهم] عبادا أذلاء، فلو كان بينهم وبينه نسب، لم يكونوا (1) كذلك.
{ سُبْحَانَ اللَّهِ } الملك العظيم، الكامل الحليم، عما يصفه به المشركون من كل وصف أوجبه كفرهم وشركهم.
{ إِلا عِبَادَ اللَّهِ الْمُخْلَصِينَ } فإنه لم ينزه نفسه عما وصفوه به، لأنهم لم يصفوه إلا بما يليق بجلاله، وبذلك كانوا مخلصين.
__________
(1) كذا في ب، وفي أ: لم يكن.
