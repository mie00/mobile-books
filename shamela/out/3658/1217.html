{ حُنَفَاءَ لِلَّهِ غَيْرَ مُشْرِكِينَ بِهِ وَمَنْ يُشْرِكْ بِاللَّهِ فَكَأَنَّمَا خَرَّ مِنَ السَّمَاءِ فَتَخْطَفُهُ الطَّيْرُ أَوْ تَهْوِي بِهِ الرِّيحُ فِي مَكَانٍ سَحِيقٍ } .
أمرهم أن يكونوا { حُنَفَاءَ لِلَّهِ } أي: مقبلين عليه وعلى عبادته، معرضين عما سواه.
{ غَيْرَ مُشْرِكِينَ بِهِ وَمَنْ يُشْرِكْ بِاللَّهِ } فمثله { فَكَأَنَّمَا خَرَّ مِنَ السَّمَاءِ } أي: سقط منها { فَتَخْطَفُهُ الطَّيْرُ } بسرعة { أَوْ تَهْوِي بِهِ الرِّيحُ فِي مَكَانٍ سَحِيقٍ } أي: بعيد، كذلك المشرك، فالإيمان بمنزلة السماء، محفوظة مرفوعة.
ومن ترك الإيمان، بمنزلة الساقط من السماء، عرضة للآفات والبليات، فإما أن تخطفه الطير فتقطعه أعضاء، كذلك المشرك إذا ترك الاعتصام بالإيمان تخطفته الشياطين من كل جانب، ومزقوه، وأذهبوا عليه دينه ودنياه.
