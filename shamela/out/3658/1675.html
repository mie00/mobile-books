{ إِنَّ السَّاعَةَ لآتِيَةٌ لا رَيْبَ فِيهَا وَلَكِنَّ أَكْثَرَ النَّاسِ لا يُؤْمِنُونَ } .
{ إِنَّ السَّاعَةَ لآتِيَةٌ لا رَيْبَ فِيهَا } قد أخبرت بها الرسل الذين هم أصدق الخلق ونطقت بها الكتب السماوية، التي جميع أخبارها أعلى مراتب الصدق، وقامت عليها الشواهد المرئية، والآيات الأفقية. { وَلَكِنَّ أَكْثَرَ النَّاسِ لا يُؤْمِنُونَ } مع هذه الأمور، التي توجب كمال التصديق، والإذعان.
