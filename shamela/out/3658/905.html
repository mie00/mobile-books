{ 32 - 34 } { اللَّهُ الَّذِي خَلَقَ السَّمَاوَاتِ وَالأرْضَ وَأَنْزَلَ مِنَ السَّمَاءِ مَاءً فَأَخْرَجَ بِهِ مِنَ الثَّمَرَاتِ رِزْقًا لَكُمْ وَسَخَّرَ لَكُمُ الْفُلْكَ لِتَجْرِيَ فِي الْبَحْرِ بِأَمْرِهِ وَسَخَّرَ لَكُمُ الأنْهَارَ * وَسَخَّرَ لَكُمُ الشَّمْسَ وَالْقَمَرَ دَائِبَيْنِ وَسَخَّرَ لَكُمُ اللَّيْلَ وَالنَّهَارَ } .
يخبر تعالى: أنه وحده { الَّذِي خَلَقَ السَّمَاوَاتِ وَالأرْضَ } على اتساعهما وعظمهما، { وَأَنزلَ مِنَ السَّمَاءِ مَاءً } وهو: المطر الذي ينزله الله من السحاب، { فَأَخْرَجَ } بذلك الماء { مِنَ الثَّمَرَاتِ } المختلفة الأنواع { رِزْقًا لَكُمْ } ورزقا لأنعامكم { وَسَخَّرَ لَكُمُ الْفُلْكَ } أي: السفن والمراكب.
{ لِتَجْرِيَ فِي الْبَحْرِ بِأَمْرِهِ } فهو الذي يسر لكم صنعتها وأقدركم عليها، وحفظها على تيار الماء لتحملكم، وتحمل تجاراتكم، وأمتعتكم إلى بلد تقصدونه.
{ وَسَخَّرَ لَكُمُ الأنْهَارَ } لتسقي حروثكم وأشجاركم وتشربوا منها.
{ وَسَخَّرَ لَكُمُ الشَّمْسَ وَالْقَمَرَ دَائِبَيْنِ } لا يفتران، ولا ينيان، يسعيان لمصالحكم، من حساب أزمنتكم ومصالح أبدانكم، وحيواناتكم، وزروعكم، وثماركم، { وَسَخَّرَ لَكُمُ اللَّيْلَ } لتسكنوا فيه { وَالنَّهَارَ } مبصرا لتبتغوا من فضله.
