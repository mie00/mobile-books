{ تَنزيلٌ مِنْ حَكِيمٍ } في خلقه وأمره، يضع كل شيء موضعه، وينزله منزله. { حَمِيدٌ } على ما له من صفات الكمال، ونعوت الجلال، وعلى ما له من العدل والإفضال، فلهذا كان كتابه، مشتملا على تمام الحكمة، وعلى تحصيل المصالح والمنافع، ودفع المفاسد والمضار، التي يحمد عليها.
[ ص 751 ]
{ 43 } { مَا يُقَالُ لَكَ إِلا مَا قَدْ قِيلَ لِلرُّسُلِ مِنْ قَبْلِكَ إِنَّ رَبَّكَ لَذُو مَغْفِرَةٍ وَذُو عِقَابٍ أَلِيمٍ } .
أي: { مَا يُقَالُ لَكَ } أيها الرسول من الأقوال الصادرة، ممن كذبك وعاندك { إِلا مَا قَدْ قِيلَ لِلرُّسُلِ مِنْ قَبْلِكَ } أي: من جنسها، بل ربما إنهم تكلموا بكلام واحد، كتعجب جميع الأمم المكذبة للرسل، من دعوتهم إلى الإخلاص للّه وعبادته وحده لا شريك له، وردهم هذا بكل طريق يقدرون عليه، وقولهم: { مَا أَنْتُمْ إِلا بَشَرٌ مِثْلُنَا }
واقتراحهم على رسلهم الآيات، التي لا يلزمهم الإتيان بها، ونحو ذلك من أقوال أهل التكذيب، لما تشابهت قلوبهم في الكفر، تشابهت أقوالهم، وصبر الرسل عليهم السلام على أذاهم وتكذيبهم، فاصبر كما صبر من قبلك.
ثم دعاهم إلى التوبة والإتيان بأسباب المغفرة، وحذرهم من الاستمرار على الغيّ فقال: { إِنَّ رَبَّكَ لَذُو مَغْفِرَةٍ } أي: عظيمة، يمحو بها كل ذنب لمن أقلع وتاب { وَذُو عِقَابٍ أَلِيمٍ } لمن: أصر واستكبر.
