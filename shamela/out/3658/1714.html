{ 16 } { وَالَّذِينَ يُحَاجُّونَ فِي اللَّهِ مِنْ بَعْدِ مَا اسْتُجِيبَ لَهُ حُجَّتُهُمْ دَاحِضَةٌ عِنْدَ رَبِّهِمْ وَعَلَيْهِمْ غَضَبٌ وَلَهُمْ عَذَابٌ شَدِيدٌ } .
وهذا تقرير لقوله: لا حجة بيننا وبينكم، فأخبر هنا أن { الَّذِينَ يُحَاجُّونَ فِي اللَّهِ } بالحجج الباطلة، والشبه المتناقضة { مِنْ بَعْدِ مَا اسْتُجِيبَ لَهُ } أي: من بعد ما استجاب للّه أولو الألباب والعقول، لما بين لهم من الآيات القاطعة، والبراهين الساطعة، فهؤلاء المجادلون للحق من بعد ما تبين { حُجَّتُهُمْ دَاحِضَةٌ } أي: باطلة مدفوعة { عِنْدَ رَبِّهِمْ } لأنها مشتملة على رد الحق وكل ما خالف الحق، فهو باطل.
{ وَعَلَيْهِمْ غَضَبٌ } لعصيانهم وإعراضهم عن حجج اللّه وبيناته وتكذيبها. { وَلَهُمْ عَذَابٌ شَدِيدٌ } هو أثر غضب اللّه عليهم، فهذه عقوبة كل مجادل للحق بالباطل.
