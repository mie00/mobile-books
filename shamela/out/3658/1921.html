{ 18-19 } { إِنَّ الْمُصَّدِّقِينَ وَالْمُصَّدِّقَاتِ وَأَقْرَضُوا اللَّهَ قَرْضًا حَسَنًا يُضَاعَفُ لَهُمْ وَلَهُمْ أَجْرٌ كَرِيمٌ } .
{ إِنَّ الْمُصَّدِّقِينَ وَالْمُصَّدِّقَاتِ } بالتشديد أي: الذين أكثروا من الصدقات الشرعية، والنفقات المرضية، { وَأَقْرَضُوا اللَّهَ قَرْضًا حَسَنًا } بأن قدموا من أموالهم في طرق الخيرات ما يكون مدخرا لهم (1) عند ربهم، { يُضَاعَفُ لَهُمُ } الحسنة بعشر أمثالها إلى سبعمائة ضعف، إلى أضعاف كثيرة، { وَلَهُمْ أَجْرٌ كَرِيمٌ } وهو ما أعده الله لهم في الجنة، مما لا تعلمه النفوس.
__________
(1) في ب: ذخرا.
