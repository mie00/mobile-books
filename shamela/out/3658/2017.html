{ 8 - 18 } { يَوْمَ تَكُونُ السَّمَاءُ كَالْمُهْلِ * وَتَكُونُ الْجِبَالُ كَالْعِهْنِ * وَلا يَسْأَلُ حَمِيمٌ حَمِيمًا } .
{ يَوْمِ } أي: القيامة، تقع فيه هذه الأمور العظيمة فـ { تَكُونُ السَّمَاءُ كَالْمُهْلِ } وهو الرصاص المذاب من تشققها وبلوغ الهول منها كل مبلغ.
{ وَتَكُونُ الْجِبَالُ كَالْعِهْنِ } وهو الصوف المنفوش، ثم تكون بعد ذاك هباء منثورا فتضمحل، فإذا كان هذا القلق والانزعاج لهذه الأجرام الكبيرة الشديدة، فما ظنك بالعبد الضعيف الذي قد أثقل ظهره بالذنوب والأوزار؟
أليس حقيقا أن ينخلع قلبه وينزعج لبه، ويذهل عن كل أحد؟ ولهذا قال: { وَلا يَسْأَلُ حَمِيمٌ حَمِيمًا * يُبَصَّرُونَهُمْ } أي: يشاهد الحميم، وهو القريب حميمه، فلا يبقى في قلبه متسع لسؤال حميمه عن حاله، ولا فيما يتعلق بعشرتهم ومودتهم، ولا يهمه إلا نفسه.
