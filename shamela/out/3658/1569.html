{ 51 - 54 } { وَنُفِخَ فِي الصُّورِ فَإِذَا هُمْ مِنَ الأجْدَاثِ إِلَى رَبِّهِمْ يَنْسِلُونَ * قَالُوا يَا وَيْلَنَا مَنْ بَعَثَنَا مِنْ مَرْقَدِنَا هَذَا مَا وَعَدَ الرَّحْمَنُ وَصَدَقَ الْمُرْسَلُونَ * إِنْ كَانَتْ إِلا صَيْحَةً وَاحِدَةً فَإِذَا هُمْ جَمِيعٌ لَدَيْنَا مُحْضَرُونَ * فَالْيَوْمَ لا تُظْلَمُ نَفْسٌ شَيْئًا وَلا تُجْزَوْنَ إِلا مَا كُنْتُمْ تَعْمَلُونَ } .
النفخة الأولى، هي نفخة الفزع والموت، وهذه نفخة البعث والنشور، فإذا نفخ في الصور، خرجوا من الأجداث والقبور، ينسلون إلى ربهم، أي: يسرعون للحضور بين يديه، لا يتمكنون من التأنِّي والتأخر، وفي تلك الحال، يحزن المكذبون، ويظهرون الحسرة والندم، ويقولون: { يَا وَيْلَنَا مَنْ بَعَثَنَا مِنْ مَرْقَدِنَا } أي: من رقدتنا في القبور، لأنه ورد في بعض الأحاديث، أن لأهل القبور رقدة قبيل النفخ في الصور، فيجابون، فيقال [لهم:] { هَذَا مَا وَعَدَ الرَّحْمَنُ وَصَدَقَ الْمُرْسَلُونَ } أي: هذا الذي وعدكم اللّه به، ووعدتكم به الرسل، فظهر صدقهم رَأْيَ عين.
ولا تحسب أن ذكر الرحمن في هذا الموضع، لمجرد الخبر عن وعده، وإنما ذلك للإخبار بأنه في ذلك اليوم العظيم، سيرون من رحمته ما لا يخطر على الظنون، ولا حسب به الحاسبون، كقوله: { الْمُلْكُ يَوْمَئِذٍ الْحَقُّ لِلرَّحْمَنِ } { وَخَشَعَتِ الأصْوَاتُ لِلرَّحْمَنِ } ونحو ذلك، مما يذكر اسمه الرحمن، في هذا.
{ إِنْ كَانَتْ } البعثة من القبور { إِلا صَيْحَةً وَاحِدَةً } ينفخ فيها إسرافيل في الصور، فتحيا الأجساد، { فَإِذَا هُمْ جَمِيعٌ لَدَيْنَا مُحْضَرُونَ } الأولون والآخرون، والإنس والجن، ليحاسبوا على أعمالهم.
{ فَالْيَوْمَ لا تُظْلَمُ نَفْسٌ شَيْئًا } لا ينقص من حسناتها، ولا يزاد في سيئاتها، { وَلا تُجْزَوْنَ إِلا مَا كُنْتُمْ تَعْمَلُونَ } من خير أو شر، فمن وجد خيرا فليحمد اللّه على ذلك، ومن وجد غير ذلك فلا يلومن إلا نفسه.
