{ 43-45 } { وَفِي ثَمُودَ إِذْ قِيلَ لَهُمْ تَمَتَّعُوا حَتَّى حِينٍ * فَعَتَوْا عَنْ أَمْرِ رَبِّهِمْ فَأَخَذَتْهُمُ الصَّاعِقَةُ وَهُمْ يَنْظُرُونَ * فَمَا اسْتَطَاعُوا مِنْ قِيَامٍ وَمَا كَانُوا مُنْتَصِرِينَ } .
أي { وَفِي ثَمُودَ } [آية عظيمة]، حين أرسل الله إليهم صالحًا عليه السلام، فكذبوه وعاندوه، وبعث الله له الناقة، آية مبصرة، فلم يزدهم ذلك إلا عتوًا ونفورًا.
فقيل { لَهُمْ تَمَتَّعُوا حَتَّى حِينٍ فَعَتَوْا عَنْ أَمْرِ رَبِّهِمْ فَأَخَذَتْهُمُ الصَّاعِقَةُ } أي: الصيحة العظيمة المهلكة { وَهُمْ يَنْظُرُونَ } إلى عقوبتهم بأعينهم.
{ فَمَا اسْتَطَاعُوا مِنْ قِيَامٍ } ينجون به من العذاب، { وَمَا كَانُوا مُنْتَصِرِينَ } لأنفسهم.
