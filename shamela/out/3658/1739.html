{ 33-35 } { وَلَوْلا أَنْ يَكُونَ النَّاسُ أُمَّةً وَاحِدَةً لَجَعَلْنَا لِمَنْ يَكْفُرُ بِالرَّحْمَنِ لِبُيُوتِهِمْ سُقُفًا مِنْ فَضَّةٍ وَمَعَارِجَ عَلَيْهَا يَظْهَرُونَ * } .
يخبر تعالى بأن الدنيا لا تسوى عنده شيئا، وأنه لولا لطفه ورحمته بعباده، التي لا يقدم عليها شيئا، لوسَّع الدنيا على الذين كفروا توسيعا عظيما، ولجعل { لِبُيُوتِهِمْ سُقُفًا مِنْ فَضَّةٍ وَمَعَارِجَ } أي: درجا من فضة { عَلَيْهَا يَظْهَرُونَ } على سطوحهم.
