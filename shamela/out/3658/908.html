ولما دعا له بالأمن دعا له ولبنيه بالأمن فقال: { وَاجْنُبْنِي وَبَنِيَّ أَنْ نَعْبُدَ الأصْنَامَ } أي: اجعلني وإياهم جانبا بعيدا عن عبادتها والإلمام بها، ثم ذكر الموجب لخوفه عليه وعلى بنيه بكثرة من افتتن وابتلي بعبادتها فقال:
[ ص 427 ]
{ 36 } { رَبِّ إِنَّهُنَّ أَضْلَلْنَ كَثِيرًا مِنَ النَّاسِ } .
أي: ضلوا بسببها، { فَمَنْ تَبِعَنِي } على ما جئت به من التوحيد والإخلاص لله رب العالمين { فَإِنَّهُ مِنِّي } لتمام الموافقة ومن أحب قوما وتبعهم التحق بهم.
{ وَمَنْ عَصَانِي فَإِنَّكَ غَفُورٌ رَحِيمٌ } وهذا من شفقة الخليل عليه الصلاة والسلام حيث دعا للعاصين بالمغفرة والرحمة من الله والله تبارك وتعالى أرحم منه بعباده لا يعذب إلا من تمرد عليه.
