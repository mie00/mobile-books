{ 12 - 14 } { إِنَّ لَدَيْنَا أَنْكَالا وَجَحِيمًا * وَطَعَامًا ذَا غُصَّةٍ وَعَذَابًا أَلِيمًا * يَوْمَ تَرْجُفُ الأرْضُ وَالْجِبَالُ وَكَانَتِ الْجِبَالُ كَثِيبًا مَهِيلا }
أي: إن عندنا { أَنْكَالا } أي: عذابا شديدا، جعلناه تنكيلا للذي لا يزال مستمرا على الذنوب. (1) { وَجَحِيمًا } أي: نارا حامية { وَطَعَامًا ذَا غُصَّةٍ } وذلك لمرارته وبشاعته، وكراهة طعمه وريحه الخبيث المنتن، { وَعَذَابًا أَلِيمًا } أي: موجعا مفظعا، وذلك { يَوْمَ تَرْجُفُ الأرْضُ وَالْجِبَالُ } من الهول العظيم، { وَكَانَتِ الْجِبَالُ } الراسيات الصم الصلاب { كَثِيبًا مَهِيلا } أي: بمنزلة الرمل المنهال المنتثر، ثم إنها تبس بعد ذلك، فتكون كالهباء المنثور.
__________
(1) في ب: على ما يغضب الله.
