{ 14 } { وَلِلَّهِ مُلْكُ السَّمَاوَاتِ وَالأرْضِ يَغْفِرُ لِمَنْ يَشَاءُ وَيُعَذِّبُ مَنْ يَشَاءُ وَكَانَ اللَّهُ غَفُورًا رَحِيمًا } .
أي: هو تعالى المنفرد بملك السماوات والأرض، يتصرف فيهما بما يشاء من الأحكام القدرية، والأحكام الشرعية، والأحكام الجزائية، ولهذا ذكر حكم الجزاء المرتب على الأحكام الشرعية، فقال: { يَغْفِرُ لِمَنْ يَشَاءُ } وهو من قام [ ص 793 ] بما أمره الله به { وَيُعَذِّبُ مَنْ يَشَاءُ } ممن تهاون بأمر الله، { وَكَانَ اللَّهُ غَفُورًا رَحِيمًا } أي: وصفه اللازم الذي لا ينفك عنه المغفرة والرحمة، فلا يزال في جميع الأوقات يغفر للمذنبين، ويتجاوز عن الخطائين، ويتقبل توبة التائبين، وينزل خيره المدرار، آناء الليل والنهار.
