{ 159 } { وَمِنْ قَوْمِ مُوسَى أُمَّةٌ } أي: جماعة { يَهْدُونَ بِالْحَقِّ وَبِهِ يَعْدِلُونَ } أي: يهدون به الناس في تعليمهم إياهم وفتواهم لهم، ويعدلون به بينهم في الحكم بينهم، بقضاياهم، كما قال تعالى: { وَجَعَلْنَا مِنْهُمْ أَئِمَّةً يَهْدُونَ بِأَمْرِنَا لَمَّا صَبَرُوا وَكَانُوا بِآيَاتِنَا يُوقِنُونَ } وفي هذا فضيلة لأمة موسى عليه الصلاة والسلام، وأن اللّه تعالى [ ص 306 ] جعل منهم هداة يهدون بأمره.
وكأن الإتيان بهذه الآية الكريمة فيه نوع احتراز مما تقدم، فإنه تعالى ذكر فيما تقدم جملة من معايب بني إسرائيل، المنافية للكمال المناقضة للهداية، فربما توهم متوهم أن هذا يعم جميعهم، فذكر تعالى أن منهم طائفة مستقيمة هادية مهدية.
