{ 10 - 13 } { وَلَقَدْ أَرْسَلْنَا مِنْ قَبْلِكَ فِي شِيَعِ الأوَّلِينَ * وَمَا يَأْتِيهِمْ مِنْ رَسُولٍ إِلا كَانُوا بِهِ يَسْتَهْزِئُونَ * كَذَلِكَ نَسْلُكُهُ فِي قُلُوبِ الْمُجْرِمِينَ * لا يُؤْمِنُونَ بِهِ وَقَدْ خَلَتْ سُنَّةُ الأوَّلِينَ } .
يقول تعالى لنبيه إذ كذبه المشركون: لم يزل هذا دأب الأمم الخالية والقرون الماضية: { وَلَقَدْ أَرْسَلْنَا مِنْ قَبْلِكَ فِي شِيَعِ الأوَّلِينَ } أي: فرقهم وجماعتهم رسلا.
{ وَمَا يَأْتِيهِمْ مِنْ رَسُولٍ } يدعوهم إلى الحق والهدى { إِلا كَانُوا بِهِ يَسْتَهْزِئُونَ }
{ كَذَلِكَ نَسْلُكُهُ } أي: ندخل التكذيب { فِي قُلُوبِ الْمُجْرِمِينَ } أي: الذين وصفهم لظلم والبهت، عاقبناهم لما اشتبهت قلوبهم بالكفر والتكذيب، تشابهت معاملتهم [ ص 430 ] لأنبيائهم ورسلهم بالاستهزاء والسخرية وعدم الإيمان ولهذا قال: { لا يُؤْمِنُونَ بِهِ وَقَدْ خَلَتْ سُنَّةُ الأوَّلِينَ } أي: عادة الله فيهم بإهلاك من لم يؤمن بآيات الله.
