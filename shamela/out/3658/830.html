{ 22 } { وَلَمَّا بَلَغَ أَشُدَّهُ آتَيْنَاهُ حُكْمًا وَعِلْمًا وَكَذَلِكَ نَجْزِي الْمُحْسِنِينَ } .
أي: { وَلَمَّا بَلَغَ } يوسف { أَشُدَّهُ } أي: كمال قوته المعنوية والحسية، وصلح لأن يتحمل الأحمال الثقيلة، من النبوة والرسالة. { آتَيْنَاهُ حُكْمًا وَعِلْمًا } أي: جعلناه نبيا رسولا وعالما ربانيا، { وَكَذَلِكَ نَجْزِي الْمُحْسِنِينَ } في عبادة الخالق ببذل الجهد والنصح فيها، وإلى عباد الله ببذل النفع والإحسان إليهم، نؤتيهم من جملة الجزاء على إحسانهم [ ص 396 ] علما نافعا.
ودل هذا، على أن يوسف وفَّى مقام الإحسان، فأعطاه الله الحكم بين الناس والعلم الكثير والنبوة.
