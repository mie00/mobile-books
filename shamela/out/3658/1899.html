{ 43-45 } { هَذِهِ جَهَنَّمُ الَّتِي يُكَذِّبُ بِهَا الْمُجْرِمُونَ * يَطُوفُونَ بَيْنَهَا وَبَيْنَ حَمِيمٍ آنٍ * فَبِأَيِّ آلاءِ رَبِّكُمَا تُكَذِّبَانِ } .
أي: يقال للمكذبين بالوعد والوعيد حين تسعر الجحيم: { هَذِهِ جَهَنَّمُ الَّتِي يُكَذِّبُ بِهَا الْمُجْرِمُونَ } فليهنهم تكذيبهم بها، وليذوقوا من عذابها ونكالها وسعيرها وأغلالها، ما هو جزاء لتكذيبهم (1) .
{ يَطُوفُونَ بَيْنَهَا } أي: بين أطباق الجحيم ولهبها { وَبَيْنَ حَمِيمٍ آنٍ } أي: ماء حار جدا قد انتهى حره، وزمهرير قد اشتد برده وقره، { فَبِأَيِّ آلاءِ رَبِّكُمَا تُكَذِّبَانِ } ولما ذكر ما يفعل بالمجرمين، ذكر جزاء المتقين الخائفين فقال:
__________
(1) في ب: جزاء لهم على تكذيبهم.
