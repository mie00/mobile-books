{ 34 } { وَلا تَقْرَبُوا مَالَ الْيَتِيمِ إِلا بِالَّتِي هِيَ أَحْسَنُ حَتَّى يَبْلُغَ أَشُدَّهُ وَأَوْفُوا بِالْعَهْدِ إِنَّ الْعَهْدَ كَانَ مَسْئُولا } .
وهذا من لطفه ورحمته تعالى باليتيم الذي فقد والده وهو صغير غير عارف بمصلحة نفسه ولا قائم بها أن أمر أولياءه بحفظه وحفظ ماله وإصلاحه وأن لا يقربوه { إِلا بِالَّتِي هِيَ أَحْسَنُ } من التجارة فيه وعدم تعريضه للأخطار، والحرص على تنميته، وذلك ممتد إلى أن { يَبْلُغَ } اليتيم { أَشُدَّهُ } أي: بلوغه وعقله ورشده، فإذا بلغ أشده زالت عنه الولاية وصار ولي نفسه ودفع إليه ماله.
كما قال تعالى: { فَإِنْ آنَسْتُمْ مِنْهُمْ رُشْدًا فَادْفَعُوا إِلَيْهِمْ أَمْوَالَهُمْ } { وَأَوْفُوا بِالْعَهْدِ } الذي عاهدتم الله عليه والذي عاهدتم الخلق عليه. { إِنَّ الْعَهْدَ كَانَ مَسْئُولا } أي: مسئولين عن الوفاء به وعدمه، فإن وفيتم فلكم الثواب الجزيل وإن لم تفوا (1) فعليكم الإثم العظيم.
__________
(1) في ب: تفعلوا.
