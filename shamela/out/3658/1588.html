{ 75 - 82 } { وَلَقَدْ نَادَانَا نُوحٌ فَلَنِعْمَ الْمُجِيبُونَ * وَنَجَّيْنَاهُ وَأَهْلَهُ مِنَ الْكَرْبِ الْعَظِيمِ } . يخبر تعالى عن عبده ورسوله نوح عليه السلام، أول الرسل، أنه لما دعا قومه إلى اللّه، تلك المدة الطويلة فلم يزدهم دعاؤه، إلا فرارا، أنه نادى ربه فقال: { رَبِّ لا تَذَرْ عَلَى الأرْضِ مِنَ الْكَافِرِينَ دَيَّارًا } الآية.
وقال: { رَبِّ انْصُرْنِي عَلَى الْقَوْمِ الْمُفْسِدِينَ } فاستجاب اللّه له، ومدح تعالى نفسه فقال: { فَلَنِعْمَ الْمُجِيبُونَ } لدعاء الداعين، وسماع تبتلهم وتضرعهم، أجابه إجابة طابق ما سأل، نجاه وأهله من الكرب العظيم.
