{ وَإِنْ يُكَذِّبُوكَ فَقَدْ كُذِّبَتْ رُسُلٌ مِنْ قَبْلِكَ وَإِلَى اللَّهِ تُرْجَعُ الأمُورُ } .
{ وَإِنْ يُكَذِّبُوكَ } يا أيها الرسول، فلك أسوة بمن قبلك من المرسلين، { فَقَدْ كُذِّبَتْ رُسُلٌ مِنْ قَبْلِكَ } فأهلك المكذبون، ونجى اللّه الرسل وأتباعهم. { وَإِلَى اللَّهِ تُرْجَعُ الأمُورُ }
