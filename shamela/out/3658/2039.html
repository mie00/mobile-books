{ 11 - 31 } { ذَرْنِي وَمَنْ خَلَقْتُ وَحِيدًا * وَجَعَلْتُ لَهُ مَالا مَمْدُودًا * وَبَنِينَ شُهُودًا * وَمَهَّدْتُ لَهُ تَمْهِيدًا * ثُمَّ يَطْمَعُ أَنْ أَزِيدَ * كَلا إِنَّهُ كَانَ لآيَاتِنَا عَنِيدًا * سَأُرْهِقُهُ صَعُودًا } .
هذه الآيات، نزلت في الوليد بن المغيرة، معاند الحق، والمبارز لله ولرسوله بالمحاربة والمشاقة، فذمه الله ذما لم يذمه (1) غيره، وهذا جزاء كل من عاند الحق ونابذه، أن له الخزي في الدنيا، ولعذاب الآخرة أخزى، فقال: { ذَرْنِي وَمَنْ خَلَقْتُ وَحِيدًا } أي: خلقته منفردا، بلا مال ولا أهل، ولا غيره، فلم أزل أنميه وأربيه (2) ، { وَجَعَلْتُ لَهُ مَالا مَمْدُودًا } أي: كثيرا { و } جعلت له { بنين } أي: ذكورا { شُهُودًا } أي: دائما حاضرين عنده، [على الدوام] يتمتع بهم، ويقضي بهم حوائجه، ويستنصر بهم.
{ وَمَهَّدْتُ لَهُ تَمْهِيدًا } أي: مكنته من الدنيا وأسبابها، حتى انقادت له مطالبه، وحصل على (3) ما يشتهي ويريد، { ثُمَّ } مع هذه النعم والإمدادات { يَطْمَعُ أَنْ أَزِيدَ } أي: يطمع أن ينال نعيم الآخرة كما نال نعيم الدنيا.
{ كَلا } أي: ليس الأمر كما طمع، بل هو بخلاف مقصوده ومطلوبه، وذلك لأنه { كَانَ لآيَاتِنَا عَنِيدًا } أي: معاندا، عرفها ثم أنكرها، ودعته إلى الحق فلم ينقد لها ولم يكفه أنه أعرض وتولى عنها، بل جعل يحاربها ويسعى في إبطالها، ولهذا قال عنه:
__________
(1) في ب: لم يذم به غيره.
(2) في ب: أربيه وأعطيه.
(3) في ب: وحصل له.
