{ جَزَاؤُهُمْ عِنْدَ رَبِّهِمْ جَنَّاتُ عَدْنٍ تَجْرِي مِنْ تَحْتِهَا الأنْهَارُ خَالِدِينَ فِيهَا أَبَدًا رَضِيَ اللَّهُ عَنْهُمْ وَرَضُوا عَنْهُ ذَلِكَ لِمَنْ خَشِيَ رَبَّهُ } .
{ جَزَاؤُهُمْ عِنْدَ رَبِّهِمْ جَنَّاتُ عَدْنٍ } أي: جنات إقامة، لا ظعن فيها ولا رحيل، ولا طلب لغاية فوقها، { تَجْرِي مِنْ تَحْتِهَا الأنْهَارُ خَالِدِينَ فِيهَا أَبَدًا رَضِيَ اللَّهُ عَنْهُمْ وَرَضُوا عَنْهُ } فرضي عنهم بما قاموا به من مراضيه، ورضوا عنه، بما أعد لهم من أنواع الكرامات وجزيل المثوبات { ذَلِكَ } الجزاء الحسن { لِمَنْ خَشِيَ رَبَّهُ } أي: لمن خاف الله، فأحجم عن معاصيه، وقام بواجباته (1) .
[تمت بحمد لله]
تفسير سورة إذا زلزلت
وهي مدنية
__________
(1) في ب: بما أوجب عليه.
