{ 65 - 66 } { أَلَمْ تَرَ أَنَّ اللَّهَ سَخَّرَ لَكُمْ مَا فِي الأرْضِ وَالْفُلْكَ تَجْرِي فِي الْبَحْرِ بِأَمْرِهِ وَيُمْسِكُ السَّمَاءَ أَنْ تَقَعَ عَلَى الأرْضِ إِلا بِإِذْنِهِ إِنَّ اللَّهَ بِالنَّاسِ لَرَءُوفٌ رَحِيمٌ * وَهُوَ الَّذِي أَحْيَاكُمْ ثُمَّ يُمِيتُكُمْ ثُمَّ يُحْيِيكُمْ إِنَّ الإنْسَانَ لَكَفُورٌ } .
أي: ألم تشاهد ببصرك وقلبك نعمة ربك السابغة، وأياديه الواسعة، و { أَنَّ اللَّهَ سَخَّرَ لَكُمْ مَا فِي الأرْضِ } من حيوانات، ونبات، وجمادات، فجميع ما في الأرض، مسخر لبني آدم، حيواناتها، لركوبه، وحمله، وأعماله، وأكله، وأنواع انتفاعه، وأشجارها، وثمارها، يقتاتها، وقد سلط على غرسها واستغلالها، ومعادنها، يستخرجها، وينتفع بها، { وَالْفُلْكَ } أي: وسخر لكم الفلك، وهي السفن [ ص 545 ] { تَجْرِي فِي الْبَحْرِ بِأَمْرِهِ } تحملكم، وتحمل تجاراتكم، وتوصلكم من محل إلى محل، وتستخرجون من البحر حلية تلبسونها، ومن رحمته بكم أنه { يُمْسِكُ السَّمَاءَ أَنْ تَقَعَ عَلَى الأرْضِ } فلولا رحمته وقدرته، لسقطت السماء على الأرض، فتلف ما عليها، وهلك من فيها { إِنَّ اللَّهَ يُمْسِكُ السَّمَاوَاتِ وَالأرْضَ أَنْ تَزُولا وَلَئِنْ زَالَتَا إِنْ أَمْسَكَهُمَا مِنْ أَحَدٍ مِنْ بَعْدِهِ إِنَّهُ كَانَ حَلِيمًا غَفُورًا } .
{ إِنَّ اللَّهَ بِالنَّاسِ لَرَءُوفٌ رَحِيمٌ } أرحم بهم من والديهم، ومن أنفسهم، ولهذا يريد لهم الخير، ويريدون لها الشر والضر، ومن رحمته، أن سخر لهم ما سخر من هذه الأشياء.
{ وَهُوَ الَّذِي أَحْيَاكُمْ } أوجدكم من العدم { ثُمَّ يُمِيتُكُمْ } بعد أن أحياكم، { ثُمَّ يُحْيِيكُمْ } بعد موتكم، ليجازي المحسن بإحسانه، والمسيء بإساءته، { إِنَّ الإنْسَانَ } أي: جنسه، إلا من عصمه الله { لَكَفُورٌ } لنعم الله، كفور بالله، لا يعترف بإحسانه، بل ربما كفر بالبعث وقدرة ربه.
