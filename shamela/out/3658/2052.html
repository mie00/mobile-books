{ 28 } ثم استدل عليهم وعلى بعثهم بدليل عقلي، وهو دليل الابتداء، فقال: { { نَحْنُ خَلَقْنَاهُمْ } أي: أوجدناهم من العدم، { وَشَدَدْنَا أَسْرَهُمْ } أي: أحكمنا خلقتهم بالأعصاب، والعروق، والأوتار، والقوى الظاهرة والباطنة، حتى تم الجسم واستكمل، وتمكن من كل ما يريده، فالذي أوجدهم على هذه الحالة، قادر على أن يعيدهم بعد موتهم لجزائهم، والذي نقلهم في هذه الدار إلى هذه الأطوار، لا يليق به أن يتركهم سدى، لا يؤمرون، ولا ينهون، ولا يثابون، ولا يعاقبون، ولهذا قال: { بَدَّلْنَا أَمْثَالَهُمْ تَبْدِيلا } أي: أنشأناكم للبعث نشأة أخرى، وأعدناكم بأعيانكم، وهم بأنفسهم أمثالهم. { إِنَّ هَذِهِ تَذْكِرَةٌ } أي: يتذكر بها المؤمن، فينتفع بما فيها من التخويف والترغيب. { فَمَنْ شَاءَ اتَّخَذَ إِلَى رَبِّهِ سَبِيلا } أي: طريقا موصلا إليه، فالله يبين الحق والهدى، ثم يخير الناس بين الاهتداء بها أو النفور عنها، مع قيام الحجة عليهم (1) ، { وَمَا تَشَاءُونَ إِلا أَنْ يَشَاءَ اللَّهُ } فإن مشيئة الله نافذة، { إِنَّ اللَّهَ كَانَ عَلِيمًا حَكِيمًا } فله الحكمة في هداية المهتدي، وإضلال الضال. { يُدْخِلُ مَنْ يَشَاءُ فِي رَحْمَتِهِ } فيختصه بعنايته، ويوفقه لأسباب السعادة ويهديه لطرقها. { وَالظَّالِمِينَ } الذين اختاروا الشقاء على الهدى { أَعَدَّ لَهُمْ عَذَابًا أَلِيمًا } [بظلمهم وعدوانهم]. تم تفسير سورة الإنسان - ولله الحمد والمنة (2) .
