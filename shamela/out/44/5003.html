{ فَأَمَّا الْيَتِيمَ فَلا تَقْهَرْ (9) وَأَمَّا السَّائِلَ فَلا تَنْهَرْ (10) وَأَمَّا بِنِعْمَةِ رَبِّكَ فَحَدِّثْ (11) }
فأما اليتيم فلا تُسِئْ معاملته، وأما السائل فلا تزجره، بل أطعمه، واقض حاجته، وأما بنعمة ربك التي أسبغها عليك فتحدث بها. { سورة الشرح }
